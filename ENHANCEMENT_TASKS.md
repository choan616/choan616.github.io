# 📖 Diary PWA - 애플리케이션 고도화 과제

이 문서는 앱의 핵심 기능이 안정화된 현시점에서, 사용자 경험을 한 단계 더 높이고 앱의 가치를 더하기 위한 향후 개발 과제를 정의하고 우선순위를 정하기 위해 작성되었습니다.

---

## 🗺️ 고도화 로드맵

### 1. 사용자 경험(UX) 고도화 (Phase 1 완료)

사용자가 매일 앱을 사용하면서 즐거움을 느끼고, 더 편리하게 일기를 관리할 수 있도록 만드는 기능들입니다.

#### 1.1 다크 모드 (Dark Mode)
- [x] **과제**: 라이트/다크 모드 전환 기능 구현
- **내용**: 시스템 설정 감지 및 설정 메뉴 토글을 통해 테마 전환. 야간 사용 시 눈의 피로 감소.
- **난이도**: 🟢 **낮음**
- **상태**: **완료** (다크모드 그림자 개선 포함)

#### 1.2 태그 필터링 및 관리
- [x] **과제**: 태그를 활용한 일기 필터링 기능 구현
- **내용**: 태그 클라우드를 통해 특정 태그가 포함된 일기를 검색.
- **난이도**: 🟢 **낮음**
- **상태**: **완료** (모바일 접기/펼치기 UI 최적화 완료)

#### 1.3 월별/연도별 통계
- [x] **과제**: 일기 작성 습관을 시각화하는 통계 대시보드 구현
- **내용**: 월별 작성 개수, 많이 쓴 태그 등을 차트로 시각화.
- **난이도**: 🟡 **중간**
- **상태**: **완료**

#### 1.4 모바일 UI 최적화
- [x] **과제**: 모바일 환경에서의 사용성 개선
- **내용**: 태그 클라우드 공간 최적화, 통계 버튼 이동 등.
- **상태**: **완료**

#### 1.5 보안 기능 강화
- [x] **과제**: 유연한 보안 설정
- **내용**: 화면 잠금 사용 여부 설정(Toggle) 추가, 다크모드 보안 UI 개선.
- **상태**: **완료**

### 2. 다음 단계 고도화 과제 (Phase 2 제안)

앱의 사용자 경험을 심화하고 데이터 활용도를 높이는 기능들입니다.

#### 2.1 🖼️ 사진 모아보기 (Image Gallery)
- [ ] **과제**: 모든 일기의 사진을 한눈에 보는 갤러리 뷰 구현
- **내용**: 연도/월별로 사진을 그룹화하여 그리드 형태로 표시, 클릭 시 해당 일기로 이동.
- **난이도**: 🟡 **중간**
- **우선순위**: ⭐⭐⭐⭐⭐ (시각적 만족도가 가장 높음)

#### 2.2 🔍 검색 기능 강화 (Enhanced Search)
- [ ] **과제**: 검색 경험 고도화
- **내용**: 검색 결과에서 매칭된 단어 하이라이팅(형광펜 효과), 기간 설정 필터 추가.
- **난이도**: � **낮음**
- **우선순위**: ⭐⭐⭐⭐

#### 2.3 📊 통계 대시보드 확장 (Extended Stats)
- [ ] **과제**: 풍부한 인사이트 제공
- **내용**: 잔디 심기(히트맵), 워드 클라우드(본문 분석), '작년 오늘' 기능 등.
- **난이도**: 🟡 **중간**
- **우선순위**: ⭐⭐⭐

#### 2.4 🌙 UX 디테일 (Micro Interactions)
- [ ] **과제**: 앱의 완성도(Polish) 높이기
- **내용**: 화면 전환 애니메이션, 스낵바 Undo, 버튼 햅틱 등 섬세한 인터랙션 추가.
- **난이도**: � **낮음**
- **우선순위**: ⭐⭐⭐
