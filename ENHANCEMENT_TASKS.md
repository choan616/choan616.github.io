### 2. 다음 단계 고도화 과제 (Phase 2 제안)

앱의 사용자 경험을 심화하고, 아키텍처를 개선하여 확장성을 높이는 기능들입니다.
이 단계에서는 사용자 경험을 크게 향상시킬 수 있는 핵심 기능 구현에 집중합니다.

#### 2.1 🚪 진입 로직 개선 및 로그인 분리 (Onboarding & Login Decoupling)
- [x] **과제**: 구글 로그인 없이 앱을 시작하고, 필요시 로그인하도록 로직 변경
- **내용**:
  1.  **앱 진입 로직 수정**: 앱 시작 시 로그인 확인 절차를 제거하고, 로컬 DB를 우선으로 사용하도록 변경합니다.
  2.  **로그인 유도 UI 구현**: '자동 동기화', '백업' 등 동기화 기능 사용 시점에 로그인 혜택을 안내하고 로그인을 유도하는 UI(모달 등)를 추가합니다.
  3.  **서비스 로직 분리**: `googleDriveService`를 호출하는 부분을 리팩토링하여, UI/UX 흐름에 따라 명시적으로 `signIn`이 호출되도록 구조를 개선합니다.
  4.  **게스트 프로필 이름 변경**: 로그인하지 않은 사용자의 프로필 이름을 '방문자' 대신 '내 다이어리'로 변경하여 소유감을 높입니다.
  5.  **PIN 설정/변경 기능**: 설정 메뉴에서 PIN을 설정하거나 변경할 수 있는 UI를 제공하고, PIN 잠금 기능을 안정화합니다.
- **기대 효과**:
  -   초기 사용자 진입 장벽 대폭 감소 및 이탈률 방지
  -   개인정보 보호를 중시하는 사용자에게 로컬 전용 사용 선택권 제공
  -   향후 Dropbox 등 다른 클라우드 저장소 확장 기반 마련
- **난이도**: 🟡 **중간**
- **우선순위**: ⭐⭐⭐⭐⭐ (핵심 아키텍처 개선 및 UX 향상)
- **상태**: **완료**

#### 2.3 📖 일기 읽기 전용 모드 UI 개선 (Read Mode UI Enhancement)
- [x] **과제**: 일기 조회 시 읽기에 최적화된 UI 제공
- **내용**: 일기 조회 시 편집기 UI 대신, 텍스트 입력창의 테두리를 없애고, 배경색을 본문과 일치시키며, 폰트나 줄 간격을 읽기 편하게 조절하여 마치 잘 편집된 블로그 글처럼 보이게 만듭니다. 이미지 레이아웃 개선 (글 중간 삽입 등)도 고려합니다.
- **난이도**: 🟡 **중간**
- **우선순위**: ⭐⭐⭐⭐⭐ (핵심 UX 향상)
- **상태**: **완료**

#### 2.4  기능 안내 온보딩 (Feature Onboarding Guide)
- [x] **과제**: 신규 사용자를 위한 주요 기능 안내 UI 구현
- **내용**:
  1.  앱 최초 실행 시 한 번만 표시되는 툴팁 또는 모달 창을 구현합니다.
  2.  '백업 및 동기화', '설정' 등 핵심 부가 기능의 위치와 역할을 간략히 안내합니다.
  3.  "다시 보지 않기" 옵션을 제공하여 사용자 경험을 해치지 않도록 합니다.
- **난이도**: 🟢 **낮음**
- **우선순위**: ⭐⭐⭐⭐ (신규 사용자 경험 및 기능 발견율 향상)
- **상태**: **완료**

- **내용**: 잔디 심기(히트맵), 워드 클라우드(본문 분석), '작년 오늘' 기능 등.

#### 2.7 🌙 UX 디테일 (Micro Interactions)
- [x] **과제**: 앱의 완성도(Polish) 높이기
- **내용**: 화면 전환 애니메이션, 버튼 인터랙션 등 섬세한 인터랙션 추가.
- **구현 내역**:
  1. **화면 전환 애니메이션**: 일기 전환 시 부드러운 fade-in-up 효과 (300ms)
  2. **버튼 마이크로 인터랙션**: 
     - Hover/Active 상태 피드백 (transform, shadow)
     - Ripple 효과 (클릭 시 물결 효과)
     - 버튼별 차별화된 스타일링 (Primary, Secondary, Danger)
  3. **특별 효과**:
     - 이미지 삭제 버튼 회전 애니메이션
     - 저장 중 pulse 애니메이션
     - Gradient 배경 (저장 버튼)
  4. **모바일 최적화**: 터치 영역 확대 (44px), 간격 조정
- **난이도**: 🟢 **낮음**
- **우선순위**: ⭐⭐⭐
- **상태**: **완료** (2025-12-19)
